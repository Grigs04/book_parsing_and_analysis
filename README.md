# Book Analysis — Анализ Частоты Слов в Книгах

**CLI-утилита для анализа текстов книг с Gutenber.org.** Программа загружает книги по URL, считает статистику частоты слов, длину книги, время чтения и позволяет интерактивно изучать результаты.

---

## Описание проекта

Этот проект анализирует тексты книг, скачанных с сайта *Project Gutenberg*.  
Он позволяет:

- Скачивать HTML книги по списку URL
- Преобразовывать текст в слова
- Считать статистику слов (общее количество, уникальные, частота)
- Сохранять результаты в SQLite-базу данных
- Выбирать книгу и смотреть статистику через консольное меню

Проект полезен тем, кто хочет:
- Исследовать тексты (частотный анализ)
- Проводить простую статистику по книгам
- Учиться работать с парсингом, БД и CLI

---

## Возможности

✔ Загрузка и парсинг книг по URL  
✔ Подсчёт общей и уникальной частоты слов  
✔ Статистика чтения (слова и примерное время)  
✔ Индивидуальный поиск слова в книге

## Описание проекта
project_root
├── config.py          # ссылки на книги, стоп-слова, путь к БД
├── db.py              # работа с БД
├── fetch.py           # парсинг HTML
├── analysis.py        # анализ текста (статистика и счётчик)
├── console.py         # консольный интерфейс
├── main.py            # точка входа, инициализация проекта
└── data/
    ├── book.db        # база данных

## Как использовать
Запустить проект (в корне проекта):
```bash
python main.py
```

Если база данных пуста, проект сам:
- Скачает книги из config.urls
- Посчитает статистику
- Сохранит всё в data/book.db
- Выбери книгу из списка.

В меню можно:
- Получить ссылку на книгу
- Посмотреть общую статистику
- Узнать, сколько раз встречается слово
- Сменить книгу
- Выйти
